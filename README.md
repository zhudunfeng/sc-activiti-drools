# sc-activiti-drools
activiti整合drools案例

主要使用的技术：spring+springboot2.0+activiti6.0+drools7.0+mybatis_plus4
<br />
本项目使用了两个案例：（1）请假案例（2）打折案例
<br />
ActivitiAndDroolsTest主要是请假案例
<br />
SaleDemoTest主要是打折案例
<br />
DroolsDemoTest主要是Drools原生api的使用
